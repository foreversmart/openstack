# openstack

Golang SDK of Openstack services.


### Development

```bash
# create a new workspace
$mkdir -p Workspaces/qiniu
$cd Workspaces/qiniu

# checkout repo
$git clone git@github.com:qbox/openstack.git

$cd openstack

# fix go path
$source env.sh

# for test
$make
```
